# 📦 運單預填工具

Invoice + Packing List PDF → DHL / FedEx / UPS 運單欄位自動抽取工具

## 🚀 使用方式

### 網上版（推薦 - 最簡單）
直接打開瀏覽器存取：
https://sssunwl.github.io/KakaTools/waybill-prefill/

### 本機版
1. 下載此資料夾
2. 雙擊 `index.html` 或用瀏覽器 File → Open 打開
3. 完全離線可用，PDF 不上傳任何地方

## 📝 使用步驟

1. **上傳 PDF** — 選擇 Invoice 和 Packing List 兩個 PDF 檔案
2. **點解析** — 工具自動抓取所有欄位
3. **複製欄位** — 逐欄位複製，貼進 DHL/FedEx/UPS 官網表單

## ✨ 特色

- ✅ 支援 DHL Express、FedEx、UPS 三家物流商欄位格式
- ✅ 自動從 Invoice + Packing List 合併資訊
- ✅ 缺少的欄位會標紅色提示（不會默默留空）
- ✅ 完全本機運行，PDF 內容不上傳伺服器
- ✅ 支援手機/平板 拖拉 PDF
- ✅ 無需安裝、無需登入

## 🔒 隱私保護

所有 PDF 解析都在您的瀏覽器本機執行。PDF 內容 **不會上傳到任何伺服器**，完全隱私。

## 💡 已知限制

- 寄件人預設為 Amphenol RF Asia Limited（香港）
- FedEx/UPS 欄位對應表基於公開文件，可能需微調
- 若 PDF 格式特殊，某些欄位可能抓不到（會標紅提示，可手動輸入）
